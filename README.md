# 📊 Sales Forecasting Dashboard

This project is a data analysis and forecasting dashboard built using Python and Streamlit.

## 🔧 Features
- Data cleaning and preprocessing
- Sales trend analysis
- Data visualization using Matplotlib
- Forecasting using ARIMA
- SQL equivalent queries for analysis

## 🛠️ Tools Used
- Python (Pandas, NumPy, Matplotlib)
- Streamlit
- Statsmodels (ARIMA)
- SQL

## 📈 Insights
- Identified sales trends over time
- Found top-performing products
- Generated future sales forecasts

## 🚀 How to Run
```bash
streamlit run app.py
